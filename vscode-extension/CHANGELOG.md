# Change Log

## 0.1.0

- Initial release.
- `clichat: Join a room` — prompts for a room number and runs `clichat <room>` in the integrated terminal.
- `clichat: Start server` — runs `clichat-server` in the integrated terminal.
- Settings: `clichat.serverUrl`, `clichat.defaultRoom`, `clichat.defaultName`, `clichat.cliPath`, `clichat.serverCliPath`.
